# Essay Generation App ✍️

A simple essay generator built with **Next.js 15**, **Tailwind CSS**, and **Google Gemini API**.

## 🚀 Features
- Generate essays using AI
- Download essays as `.pdf`
- Modern UI with Tailwind CSS

## ⚙️ Installation
```bash
git clone <your-repo-url>
cd essay-generation
npm install
npm run dev
